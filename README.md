# English Word Learning Frontend

This is the frontend application for the English Word Learning software, built with Vue.js.

## Features

1. **Video Subtitle Generation & Translation**
   - Upload videos and generate subtitles
   - Display Chinese and English subtitles simultaneously
   - Smart duplicate detection based on file hash

2. **Video History Management**
   - View all processed videos
   - Quick access to historical subtitles
   - Video deletion and management

3. **Planned Vocabulary Learning** ‚≠ê **NEW**
   - **Step 1**: Enhanced subtitle display with word highlighting
   - **Step 2**: AI-generated comprehension questions
   - **Step 3**: Story generation using learned words
   - **Step 4**: Memory testing and progress tracking
   - Complete learning session with statistics

4. **Word Book Management**
   - Select words from subtitles to add to personal word book
   - Manage (view, delete, categorize) words in the word book
   - Batch operations (move, delete multiple words)
   - Search and filter functionality

5. **Base Vocabulary Library**
   - 14,713 high-quality English words
   - Local-first querying with intelligent fallback
   - Search and difficulty-based filtering
   - Vocabulary coverage analysis

6. **Story Generation**
   - Select words from word book to generate stories
   - View stories in both English and Chinese
   - Word highlighting in generated stories
